■これは何をするものか

Discordのゲーム中表示にゴーストの名前を入れます。

■動作環境

・Windows 7以降

■使い方

インストールしたら勝手に動きます。
「開発中」フラグを設定したい場合は、プラグインメニューからDiscord Rich Presenceを選んで下さい。

■配布条件等

license.txtを見てください。たいしたことは書いてないです。

■特殊アイコンについて

一部特殊なアイコンに切り替わるゴーストさんがいますが、これは本プラグインやそれ以外で何かContribute
していただいた方へのささやかなお礼になります。お楽しみください。
